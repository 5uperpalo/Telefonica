# Telefonica project

Scripts related to "Socioeconomic Fairness in Mobile Network: A Close Examination of the Infrastructure and Performance" project done during the stay at Telefonica :

 * “Data scientist (fixed-term 3 months) at O2 Telefonica Barcelona, Spain”
21/11/2018 – 20/02/2019
Analysis of the relations between socioeconomic status of customers and network performance, and investigation of potential discrimination in network deployment and management. Finding the correlation between UK socioeconomic information from LSOA database (Lower-layer Super Output Areas) and operator measurements by use of Geographic Information System (QGIS, ArcGIS, GeoPandas) and distributed computing (PySpark) tools.
